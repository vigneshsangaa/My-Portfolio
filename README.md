## Featured Projects

### ADPLL — All-Digital Phase-Locked Loop
Designed and implemented at NIT Warangal during a summer internship.
- Delay-line Time-to-Digital Converter (TDC) with 100 ps resolution, 16×4 priority encoder
- 25th-order FIR loop filter with ANN-derived Hamming-windowed coefficients (Verilog-A)
- 2.26 GHz Digitally Controlled Oscillator (DCO) using PMOS/NMOS delay cells, UMC 180nm
- ÷64 TSPC frequency divider

### CardioEdge — ECG Arrhythmia Detector
1D CNN-based arrhythmia classifier trained on the MIT-BIH dataset, with RTL implementation targeting FPGA deployment (Digilent Atlys board, Xilinx Spartan-6).
- Verilog RTL design with custom BRAM/DSP resource architecture
- Resolved BRAM depth overflow via dual half-instance memory splitting
- Ported DSP primitives from Artix-7 to Spartan-6


### UART— RTL-to-GDSII

Verilog implementation of a UART transceiver — baud rate generator, TX/RX FSMs, parity handling, and loopback testbench verification.

Full physical design flow for an asynchronous FIFO, targeting the Cadence Genus/Innovus toolchain.
